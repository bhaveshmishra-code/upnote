# App Script Essentials

## **How to get date string?**

```
function getDateToday() {
  const now = new Date(Date.now())
  const year = now.getFullYear()
  const month = now.getMonth()
  const date = now.getDate()
  const today = `${date}/${month + 1}/${year}`
  return today
}
```
